# Cross-Chain Aave Rebalancer Powered by EIL Protocol

This project is a cross-chain Aave rebalancer powered by EIL Protocol. It allows you to rebalance your Aave positions across multiple chains using a single transaction.

## Features

- Rebalance Aave positions across multiple chains
- Use EIL Protocol to execute the rebalance.
